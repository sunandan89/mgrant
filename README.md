# mGrant — Donor Features

Public-facing feature site for **mGrant**, the grant-management platform for CSR teams, foundations, and grant-making organisations in India.

**Live site (once pushed):** https://sunandan89.github.io/mgrant-features/

## What's here

- `index.html` — The full single-page feature site
- `README.md` — This file

## Deploy to GitHub Pages

1. Create a new public repo named `mgrant-features` on your GitHub account.
2. Copy `index.html` and `README.md` to the repo root.
3. Commit and push to `main`.
4. Go to **Settings → Pages** and enable Pages on the `main` branch, root folder.
5. The site goes live at `https://sunandan89.github.io/mgrant-features/` within a minute or two.

## Tech

- Pure single-file HTML + CSS (no build step, no framework)
- Inter typography via Google Fonts
- Fully responsive — works on desktop, tablet, and mobile
- Dhwani maroon (`#8B1A1A`) + orange (`#E67E22`) palette — consistent with the mGrant sales deck
